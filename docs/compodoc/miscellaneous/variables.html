src/app/interceptors/client-angular.interceptor.ts

clientAngularInterceptor
Type : HttpInterceptorFn
Default value : (req, next) => { return next(req.clone({ url: `${environment.baseUrl}/${req.url}`})); }

Interceptor HTTP que concatena la URL base a todas las peticiones salientes.

Este interceptor se encarga de anteponer el baseUrl definido en el archivo de configuración de entornos (environment) a cada request HTTP.

⚠️ Nota:

  • No debe usarse con URLs absolutas (http://, https://)
  • Está pensado para trabajar con rutas relativas como /users, /login, etc.
